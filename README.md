🎉 Birthday Universe App 💖

A beautiful interactive birthday surprise web application built using Streamlit, Python, and CSS animations.
This project is designed to create a memorable, emotional, and aesthetic birthday experience with music, animations, messages, and photo memories.

💖 Live Features

✨ This app includes 5 interactive tabs:

🏠 Home Tab
Beautiful glowing birthday welcome screen
Animated aesthetic design
Personalized birthday wishes
Floating hearts & balloons animation
❤️ Heart Tab
3D animated heart design
Name displayed inside glowing heart 💖
Pulsing animation effect
Background birthday music
💌 Letter Tab
Emotional birthday letter
Personalized messages for special friend/crush
Aesthetic glassmorphism design
Romantic + appreciation tone
📸 Memories Tab
12-image memory gallery system
Auto rotating slideshow
Floating grid gallery
Manual image viewer
Sliding animated photo frames (carousel style)
Background music support 🎵
🎉 Celebration Tab
Big animated “HAPPY BIRTHDAY” text
Fireworks animation 🎆
Floating balloons 🎈
Final birthday message screen
Auto-play music support
🛠️ Tech Stack
Python 🐍
Streamlit 🎈
HTML + CSS (custom styling)
JavaScript (for animations)
NumPy & Pandas# frnds-app
